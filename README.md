# Cursor Command Generator

A simple web app to generate Cursor command links.

## Usage

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Input field for command name
- Textarea for command content
- Automatically generates the Cursor command link
- Copy button to copy the generated link to clipboard
